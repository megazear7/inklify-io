# [Inklify IO](https://inklify.io/)

This is a documentation website for Inklify. Read about Inklify and use it to write books at [inklify.io](https://inklify.io/)

## Serve

```
npm run serve
```

Open [localhost:3001](http://localhost:3001)

## Deploy

Push or merge changes to `master` and Netlify will build and deploy the changes.
